# Case Study: IT Operations Automation

## Problem

Recurring IT operations tasks create manual load:

- status checks
- log review
- documentation maintenance
- error analysis
- security hygiene
- portfolio evidence

## Solution

Combination of:

- Bash / Python automation
- Markdown runbooks
- smoke tests
- secret scanning
- GitHub documentation
- Local AI-assisted structuring
- human review before publication

## CV Bullet

Developed local IT operations automations with Bash/Python, Local AI, RAG-assisted documentation, runbooks, security scans and GitHub-based evidence to improve transparency, reproducibility and operational stability.
