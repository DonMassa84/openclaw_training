# Runbook: OpenClaw / Local AI Operations

## Purpose

This runbook describes a safe, reproducible workflow for local AI-assisted IT operations documentation.

## Standard Workflow

1. Identify recurring IT operations case.
2. Sanitize input.
3. Structure notes into Markdown.
4. Run governance and secret check.
5. Add risk/fallback section.
6. Commit portfolio-ready artifact.

## Control Points

Before publication:

- [ ] no secrets
- [ ] no private personal data
- [ ] no authority, health or finance details
- [ ] purpose and value are understandable
- [ ] security boundaries described
- [ ] GitHub README updated
- [ ] smoke test documented
